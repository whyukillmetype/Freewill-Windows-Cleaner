Distributor Guide for Freewill Program
    Program Description

    Freewill is a program written in Python designed to clean various types of data on your device.
    The program supports only the Windows operating system.

    Program Structure

    Program Modules: All program modules are saved as .pyw files. These files are intended to run without 
    opening a console window.
    Main Script: The main script is saved as a .py file. This Python file 
    contains the core code of the program and must be executed to run the program.
    Python Version: The program is written and tested using Python version 3.12.4.

    Installation and Setup

    Python: Ensure that Python version 3.12.4 is installed on the target machine. 
    The program depends on this version to function correctly.
    
    Creating the Distribution

    Package all .pyw modules and the main .py script into an archive (e.g., a .zip file).
    Ensure that when creating the distribution, you account for all exceptions related 
    to paths to system folders to avoid deployment errors.
    
    System Folder Paths

    When developing and testing, check the paths to system folders (e.g., Startup folder) 
    to avoid access errors or missing files.
    
    Running the Program

    Run the main .py script using Python 3.12.4. Ensure that all dependencies are installed and correctly configured.
    .pyw modules will automatically run without a console window.
    
    Features

    Compatibility: The program is intended for Windows OS only. 
    Ensure it is tested in a Windows environment to avoid compatibility issues.
    Exceptions: Always use exception handling when working with system folder paths and file systems 
    to ensure the reliable operation of the program and correct error handling.

    Contact Information

    If you have any questions or issues with distributing the program, you can contact the author:

    Описание Программы

    Freewill — это программа, написанная на языке Python, предназначенная для очистки различных типов данных на вашем устройстве. 
    Программа поддерживает только операционную систему Windows.

    Структура Программы

    Модули Программы: Все модули программы сохранены как .pyw файлы. 
    Эти файлы предназначены для выполнения без открытия консольного окна.

    Основной Скрипт: Основной скрипт сохранен как .py файл. Это Python-файл, 
    который содержит основной код программы и должен быть запущен для выполнения программы.

    Версия Python: Программа написана и протестирована с использованием Python версии 3.12.4.

    Установка и Настройка

    Python: Убедитесь, что на целевой машине установлена версия Python 3.12.4. 
    Программа зависит от этой версии, чтобы корректно работать.

    Создание Дистрибутива:

    Упакуйте все .pyw модули и основной .py скрипт в архив (например, .zip файл).
    Убедитесь, что при создании дистрибутива вы учитываете все исключения, 
    связанные с путями к системным папкам, чтобы избежать ошибок при развертывании.

    Пути к Системным Папкам:

    При разработке и тестировании, проверяйте пути к системным папкам (например, папка Startup), 
    чтобы избежать ошибок доступа или отсутствующих файлов.

    Запуск Программы

    Запустите основной .py скрипт с помощью Python 3.12.4. 
    Убедитесь, что все зависимости установлены и правильно настроены.
    Модули .pyw будут автоматически выполняться без консольного окна.

    Особенности

    Совместимость: Программа предназначена только для Windows OS. 
    Убедитесь, что она тестируется в среде Windows, чтобы избежать проблем совместимости.

    Исключения: Всегда используйте исключения (exception handling) 
    при работе с путями к системным папкам и файловой системой, 
    чтобы обеспечить надежную работу программы и корректную обработку ошибок.
